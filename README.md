# Modelo Predictivo para la Recomendación de Oportunidades Laborales en el Ámbito Gubernamental

## Descripción

Este proyecto hace parte de la Maestría en Ciencia de Datos y tiene como objetivo diseñar un modelo predictivo que permita recomendar oportunidades laborales del sector público colombiano a partir de la información académica, laboral y del historial de participación de los aspirantes en procesos de selección.

Actualmente el proyecto se encuentra en fase de desarrollo y comprende la construcción de la Base Analítica (ABT) y el Análisis Exploratorio de Datos (EDA), etapas fundamentales para el entrenamiento posterior de modelos de Machine Learning.

Actualmente se encuentran desarrollados:

Las siguientes etapas corresponden al preprocesamiento, ingeniería de características y entrenamiento de modelos predictivos.
